System Test Plan

Project: Inventory and Booking System
Date: 2025-12-17
Version: 2.0

1. Introduction

This document outlines the test strategy and test cases for the Inventory and Booking System. It covers functional testing of critical modules including Authentication, Inventory, Services, Bookings, Reports, and System Utilities.

2. Test Scope

  • In Scope:
    • User Authentication & Role-Based Access Control (RBAC)
    • Public Booking Portal
    • Admin/Manager Dashboard
    • Inventory Management (Items, Categories, Stock In/Out)
    • Service Management (Job Orders, Status Workflow)
    • Employee & Supplier Management
    • Reports & Analytics
    • Backup & Restore Functionality
  • Out of Scope:
    • Performance/Load Testing
    • Security Penetration Testing (handled separately)

3. Test Cases

3.1 Authentication & Security

ID Test Case Steps Expected Result Priority
TA-01 Admin Login 1. Go to /login.
2. Enter Admin credentials.
3. Submit.		 Redirect to /system Dashboard. High
TA-02 Invalid Login 1. Go to /login.
2. Enter wrong password.		 Error "Invalid credentials" displayed. High
TA-03 Manager Access Restriction 1. Login as Manager.
2. Try to access /employees (Admin only).		 Access Denied (403) or Redirected. High
TA-04 Logout 1. Click "Logout". Redirect to login page. Session cleared. Medium
